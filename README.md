# 🚚 TransportOps - Fleet Management System

A full-stack MERN Fleet Management System developed to efficiently manage vehicles, drivers, trips, maintenance, fuel expenses, and analytics through a modern dashboard.

---

## 📌 Features

### 🔐 Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Logout

### 📊 Dashboard
- Live Statistics
- Total Vehicles
- Total Drivers
- Active Trips
- Fuel Expenses
- Maintenance Summary
- Recent Trips

### 🚚 Vehicle Management
- Add Vehicle
- Update Vehicle
- Delete Vehicle
- Vehicle Status
- Search Vehicles

### 👨‍✈️ Driver Management
- Add Driver
- Edit Driver
- Delete Driver
- License Details
- Driver Availability

### 🛣 Trip Management
- Create Trip
- Assign Driver
- Assign Vehicle
- Update Trip Status
- Delete Trip

### 🔧 Maintenance Management
- Schedule Maintenance
- Update Service Status
- Track Service Cost
- Next Service Date

### ⛽ Fuel & Expense Management
- Add Fuel Logs
- Fuel Quantity
- Fuel Cost
- Odometer Reading
- Fuel Station
- Expense Tracking

### 📈 Analytics
- Fleet Analytics Dashboard
- Vehicle Status Chart
- Trip Status Chart
- Monthly Trips
- Fuel Cost Analysis
- Maintenance Cost Analysis

### ⚙️ Settings
- Update Profile
- Change Password
- Preferences
- Logout
